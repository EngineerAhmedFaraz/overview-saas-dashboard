# Overview - Modern SaaS Admin Dashboard

A clean, modern, and fully functional SaaS Admin Dashboard built with React and TypeScript. Designed to demonstrate senior-level frontend development skills.

## 🎯 Key Features

- Modern & fully responsive design
- Dark / Light mode with persistence
- Collapsible sidebar (full ↔ icon-only)
- Dynamic user profile & avatar
- Advanced data tables with search, sort & pagination
- Beautiful charts using Recharts
- Protected routes with authentication
- Complete auth flow (Login, Signup, Forgot Password)
- Settings page with Appearance tab
- Logout confirmation modal
- Professional UI using shadcn/ui + Tailwind

## 🛠️ Tech Stack

- **React 18** + **TypeScript**
- **Vite** (with Rolldown for faster builds)
- **Tailwind CSS** + **shadcn/ui**
- **TanStack Table** & **TanStack Query**
- **Recharts** for data visualization
- **Zustand** for state management
- **React Router v6** for routing
- **Zod** + **React Hook Form** for validation
- **Lucide React** for icons
